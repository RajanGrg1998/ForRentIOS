# Rent Finder iOS App

## 📱 Overview

A Swift-based iOS application that helps users find available rental properties on an interactive map.

## ✨ Features

* Map view with rental locations
* Property details display
* Location-based search
* Smooth mobile UI

## 🛠 Tech Stack

* Swift
* UIKit / SwiftUI
* MapKit
* REST APIs

## 🚀 Future Improvements

* Filters (price, location)
* User login
* Save favorite listings
